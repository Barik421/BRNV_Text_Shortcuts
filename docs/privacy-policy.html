Privacy Policy — BRNV Text Shortcuts

Effective date: 2026-04-06

BRNV Text Shortcuts (“the Extension”) is a Chrome extension that helps users create text shortcuts and expand them into longer text while typing.

Data the Extension stores

The Extension stores user-created shortcut data (such as triggers and replacement text) and extension settings (for example folders, enabled/disabled shortcuts, and disabled domains) using Chrome storage in the user’s browser profile. If Chrome Sync is enabled, Chrome may synchronize this stored data across the user’s devices.

What the Extension does NOT do

The Extension does not collect browsing history, passwords, payment information, or personal identifiers. It does not transmit stored shortcuts to external servers, and it does not sell or share data with third parties.

How the Extension works on webpages

To expand shortcuts, the Extension interacts with editable text fields only to detect the typed trigger and insert the saved replacement text. The Extension does not record keystrokes for analytics and does not send typed content anywhere.
